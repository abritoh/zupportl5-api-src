Metric Explanation

  • Afferent Couplings (Ca): The number of other packages relying on the package's classes. Is an indicator of the package's responsibility.
  • Efferent Couplings (Ce): The number of other packages that the classes in the package depends on. Is an indicator of the package's independence.
  • Instability (I 0 to 1): I = Ce / (Ce+ Ca). 0 (Zero) no other packages depend on it (highly resilient to changes). 1 (One) this means the package relies heavily on other packages.
  • Cycles: A cycle means that two or more packages depend on each other either directly or indirectly.
